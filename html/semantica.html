Etiqueta a
Etiqueta para hacer referencia a otras páginas web o recursos como documentos o archivos.
Atributo :href Valor: URL Descripcion: Enlace al documento que se quiere cargar. Atributo obligatorio.
Atributo: downlad Valor: nombre.ext Descripcion: Descarga el enlace (href) en lugar de abrirlo. Si se indica valor, se renombra
Atributo: target Valor: _blank Descripcion: Abre el enlace en una nueva pestaña. Valor:_self Abre el enlace en la pestaña o -iframe- actual.
Valor: _parent Descripcion: Abre el enlace en el documento padre.
Valor: _top Descripcion: Abre el enlace en el documento raíz (padre global).
Valor: nombre Descripcion: Abre el enlace en el -iframe- con el nombre especificado.
Equiteta rel
Atributo: rel Valor: alternate Descripcion: Indica que el enlace es una versión alternativa (idioma o formato diferente).
Valor: author Descripcion: Indica que el enlace es la web del autor de la página actual.
Valor: bookmark Descripcion: El enlace es un permalink a la sección de la página actual.
Valor: help Descripcion: Indica que el enlace ofrece ayuda para la página actual.
Valor: license Descripcion: Indica que la página actual está cubierta por la licen referenciada.
Valor: prev Descripcion: Indica que el enlace es la parte previa del documento actual.
Valor: next Descripcion: Indica que el enlace es la parte siguiente del documento actual.
Valor: nofollow Descripcion: Indica que el enlace no está supervisado por el autor del sitio web.
Valor: noreferrer Descripcion: El navegador no envía la página de procedencia al visitar el enlace.
Valor: prefetch Descripcion: El navegador precarga el enlace (es probable que se haga clic)
Valor: search Descripcion: Indica que el enlace es una página para realizar búsquedas.
Valor: tag Descripcion: Asocia una etiqueta al enlace especificado, en relación al documento actual.
Idioma Código ISO 639-1 del idioma del documento.
Tipo Tipo de contenido MIME al que se enlaza. Atributo no obligatorio.
